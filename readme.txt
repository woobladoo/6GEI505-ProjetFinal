README.md for 6GEI311 - Lab4

1. Introduction Doxygen
Doxygen est un logiciel qui permet de créer un html/interface à partir d'une documentation 
dans un code afin d'avoirdes déscriptions du fonctionnement du code et de le visualiser de 
manière intuitive et claire

2. Apercu du projet et du code
Le projet documenté est le __main__.py d'un serveur flask. Nous etions malheureusement dans l'impossibilité de terminer
la laboratoire 3 du cours 6GEI311 donc nous avons utiliser un projet en cours à documenter. Les fonctions documenté correspondent à
des fonctions de parcour de base de données et des fonctions d'affichage de page HTML.

3. Explication des commentaires générés
Les commentaires ne sont pag générés mais plutot organisé de manière claire et précis sur un interface html.
Les commentaires affiché explique les différentes fonction et parametres  à l'aide des décorateurs @brief, @param, @return.
Doxygen offre aussi un navigateur pour facilement naviguer les commentaires des fonctions et des variables. 

4. Conclusion sur l'importante de la documentation, différence entre documentation Doxygen et documentation traditionnel.
La documentation est très importante pour permettre plusieurs personnes à travailler sur un même code et se comprendre. La documentation
sur Doxygen permet de faciliter le processus de documentation des différentes fonctions afin qu'il soit plus facile de les comprendre que si 
ils étaient documenter que dans le code avec des commentaires.
